Bootstrap E-commerce Website

This is a responsive e-commerce website built using HTML5 and Bootstrap 5.  
The project is designed for beginners to understand layout structure, components, and responsive design.

Features
- Responsive Navbar
- Home, About, Contact pages
- Login & Signup pages
- Product listing section
- Categories sidebar
- FAQ section
- Fully responsive design

 Technologies Used
- HTML5
- Bootstrap 5

 Project Purpose
This project is created for learning and practice purposes.  
It helps beginners understand how to build a clean and modern e-commerce layout using Bootstrap.
